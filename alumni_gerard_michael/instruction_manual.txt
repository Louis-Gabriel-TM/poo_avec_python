

mode d'emploi:

1/ vous ètes un étudiant :
	+ appuyez sur ENTRER
	+ si vous souhaitez consulter une fiche élève appuyez sur CONSULTER
		+ choisir la fiche en cliquant sur le bouton OK à droite  du nom / prénom de l'étudiant
		+ si vous souhaitez voir une autre fiche appuyez sur CONSULTER
		+ si vous souhaitez créer votre fiche personnelle appuyez sur CREER
		+ sinon appuyer sur SORTIE
	+ si vous souhaitez créer votre fiche personnelle appuyez sur le bouton CREER
		+ ATTENTION vous ne pouvez créer qu'une seule fiche personnelle
		+ renseigner les champs et appuyer sur VALIDER (renseigner les champs avec * est obligatoire )
		+ LES INFORMATIONS ONT ETE ENREGISTREES 
		+ appuyer sur CONSULTER ou SORTIE

2/ vous ètes Mister M. :
	+ veuillez entrer votre Mot De Passe puis appuyer sur ENTRER
	+ si vous souhaitez consulter une fiche appuyez sur CONSULTER
		+ choisir la fiche en cliquant sur le bouton OK à droite  du nom / prénom de l'étudiant
		+ appuyer sur CONSULTER ou SORTIE ou MODIFIER ou SUPPRIMER ou CREER

	+ si vous souhaitez modifier une fiche appuyez sur MODIFIER
		+ choisir la fiche en cliquant sur le bouton OK à droite  du nom / prénom de l'étudiant
		+ renseigner les champs et appuyer sur VALIDER (renseigner les champs avec * est obligatoire )
		+ LES INFORMATIONS ONT ETE ENREGISTREES 
		+ appuyer sur CONSULTER ou SORTIE ou MODIFIER ou SUPPRIMER ou CREER

	+ si vous souhaitez supprimer une fiche appuyez sur SUPPRIMER
		+ choisir la fiche en cliquant sur le bouton OK à droite  du nom / prénom de l'étudiant
		+ appuyer sur VALIDER
		+ appuyer sur CONSULTER ou SORTIE ou MODIFIER ou SUPPRIMER ou CREER

	+ si vous souhaitez créer une fiche appuyez sur CREER
		+ renseigner les champs et appuyer sur VALIDER (renseigner les champs avec * est obligatoire )
		+ LES INFORMATIONS ONT ETE ENREGISTREES 
		+ appuyer sur CONSULTER ou SORTIE ou MODIFIER ou SUPPRIMER ou CREER
